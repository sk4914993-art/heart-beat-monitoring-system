# Heart Beat Monitoring System

## Overview
Arduino based system to monitor heart rate and temperature
using Pulse Sensor and LM35.

## Components Used
- Arduino UNO
- Pulse Sensor
- LM35
- LCD Display
- Jumper Wires

## Features
- Real-time BPM monitoring
- Temperature display
- LCD output

## How to Run
1. Connect hardware as per circuit.
2. Upload heartbeat.ino to Arduino.
3. Power ON the system.
